Ce document présente le projet web intitulé « Éditeur de Pixel Art et Morpion », dont le code source est disponible sur notre dépôt GitHub public.
Il s'agit d'une application web développée exclusivement en HTML, CSS et JavaScript. Elle regroupe deux jeux interactifs accessibles grâce à un menu de navigation (le morpion et le pixel art)
Le premier jeu est un éditeur de pixel art. Il y a plusieurs outils de dessin tels qu'un crayon, une gomme, un outil de remplissage (pot de peinture) et une pipette. L'utilisateur peut redimensionner la grille de travail s’il le souhaite, augmenter ou diminuer l'épaisseur de son trait et enfin sélectionner des couleurs précises grâce à une palette de couleur intégrée.
Le second jeu est morpion permettant à l'utilisateur d'affronter une intelligence artificielle. L'algorithme analyse la grille en temps réel et identifie comment gagner, bloquer les mouvements du joueur ou jouer de manière aléatoire en l'absence de menace directe (afin de laisser des chances de victoire du côté joueur). Le système détecte automatiquement l'issue de la partie et propose une option de réinitialisation afin de pouvoir enchaîner les parties.
L'ensemble de l'interface a été structuré avec un thème sombre. Ce programme s'exécute entièrement côté client et ne demande aucune installation de serveur ni quoi que ce soit d’autre.
Pour lancer le projet, il suffit de cloner le dépôt Git ou de télécharger les fichiers sources. Ensuite, il ne vous reste plus qu'à ouvrir le fichier index.html dans un navigateur web pour utiliser l'application.
